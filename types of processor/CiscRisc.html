RISC

Stands for Reduced Instruction Set Computer

Often used in portable devices such as mobiles or tablets

Is limited to only one instruction per clock cycle

Has a smaller instruction set

Compiler has to do less work

Is far away from high-level languages

CISC

Stands for Complex Instruction Set Computer

Has a large instruction set

Often used in more traditional, desktop computers

Can execute more than one instruction per clock cycle

Compiler has to do more work

Compiler has to do more work

Is close to high-level languages
